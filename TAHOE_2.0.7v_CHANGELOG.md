# Adventure 2 v2.0.7v — The Great Tahoe Rebuild

## Real exploration loop
- Expanded from one Tahoe Memory to four
- Each memory has its own icon, title, and personal story
- Discovery progress is saved and restored
- HUD now tracks 0/4 memories

## Rigsby exploration ability
- Rigsby senses nearby hidden memories
- An alert appears above him as the player gets close
- A brief companion hint helps without revealing the exact location

## Named world areas
- Emerald Shore
- Whispering Pines
- Eagle Creek
- Granite Crown
- Infested Trailhead
- Each area receives an arrival banner and updates the location HUD

## Major Tahoe landmarks
- Emerald cove
- Giant fallen pine
- Eagle Creek waterfall
- Historic ranger post ruins
- Granite dome
- Eagle Overlook

## Living-world pass
- Gliding eagle
- Dragonflies around the water
- Existing squirrels, rabbits, butterflies, ducks, fish, and birds retained
- Wildlife reacts to the player
