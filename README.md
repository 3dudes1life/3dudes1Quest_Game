# 3Dudes1Quest v0.9.2 — Definitive SoCal Playtest Build

## The major Level 1 upgrade
- Four restoration missions across Home Base, Beach, PCH/Hollywood, and Los Angeles
- Visible beige overlays disappear when each zone is restored
- Mission HUD with current objective and instructions
- Full-health checkpoint rewards after each restored zone
- Expanded rooftop and elevated secret routes
- Secret Rooftop Flamingo souvenir
- More distinct enemy behavior:
  - Trolls chase nearby players
  - Scooters move faster
  - HOA drones hover
- Stronger hero identities:
  - Will gets a rainbow dash and piercing projectile
  - Daniel's positivity shield restores one heart
  - Caleb's cookies deal heavier boss damage
- Rigsby can rescue the player once during the opening route
- Boss attacks now telegraph before firing
- Boss arena warning flash improves readability
- Victory restoration flash
- Objectives page added to the Adventure Journal
- Expanded final rank scoring
- Dedicated v0.9.2 save slot
- Music remains intentionally deferred

## Playtest focus
Please test movement, jumping routes, checkpoints, dialogue interruptions, save/continue, each character power, all four zone objectives, collectibles, the secret rooftop route, mobile buttons, boss phases, and the completion screen.
