# 3Dudes1Quest v0.9.1 — SoCal Fire Pass

This update polishes the first adventure before Lake Tahoe begins.

## Implemented
- Route progress meter with named areas
- Three unlockable selfie memories
- Memories page in the Adventure Journal
- Hidden Chaos Bowl with a timed super-jump modifier
- Secrets page in the Adventure Journal
- Funny ability text for every Gay Card
- Ambient clouds, birds, and butterflies
- Additional walking banter
- Smarter Rigsby collectible-sniff behavior
- Selfie camera flash effect
- End-of-level performance rank
- Expanded completion statistics
- Separate v0.9.1 save slot
- No music yet, by design

## Level completion target
Collect six Gay Cards, restore three Prism Beacons, discover the memories and secret, then defeat the HOA Queen of Beige.
