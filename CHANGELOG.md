# Changelog

## v1.0.3V — Responsive UX Rebuild

- Fixed the game canvas failing to fill wide desktop windows.
- Removed the exposed second-scene strip on the right side.
- Rebuilt environment rendering to cover the full canvas at any aspect ratio.
- Added responsive high-DPI canvas resizing.
- Added 16:9 desktop and mobile scaling.
- Reworked foreground environment scaling and reduced oversized plant overlays.
- Improved camera framing for wider screens.
- Reduced player, Rigsby, and boss scale for more natural world proportions.
- Repositioned HUD panels based on the active viewport width.
- Updated all screen flashes, lighting, and cinematic effects to use the real canvas dimensions.
- Added v1.0.3 cache busting.
