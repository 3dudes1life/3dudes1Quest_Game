# Adventure 2 v2.0.6 — Tahoe Playable

## Critical gameplay repair
- Replaced Tahoe's overlapping pointer/mouse/touch handlers with the proven Level One pointer system
- Added reliable keyboard capture
- Movement: Arrow keys or A / D
- Jump: Arrow Up, W, or Space
- Power: X or F
- Input safely releases when the browser loses focus

## Level One control deck
- Tahoe now uses the same five-control arrangement:
  - Left
  - Right
  - Switch
  - Power
  - Jump
- Matching grouped glass control pods
- Matching purple Switch, pink Power, and teal Jump buttons
- Visible pressed states

## Beetle combat
- Beetles can still be defeated by jumping on them
- New rainbow Power blast can destroy beetles from a distance
- Blast has visible rainbow energy and a short cooldown

## Tahoe scenic pass
- Clear shallow-water layer
- Underwater shoreline stones
- Brighter animated shoreline waves
- Curved sandy shoreline
- More varied pine silhouettes and proportions
