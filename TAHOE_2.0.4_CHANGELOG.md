# Adventure 2 v2.0.4 — Playable Input Fix

## Critical fixes
- Rebuilt keyboard input using document-level capture for Safari
- Added both `key` and `code` detection
- Added J as an additional jump key
- Rebuilt on-screen controls for mouse, trackpad, pointer, and touch
- Removed Safari blur behavior that could silently pause the game
- Controls now visibly depress when activated
- Added an input-status indicator confirming that movement was received
- Desktop controls are now fully visible instead of appearing disabled

## Visual improvement
- Replaced the unreadable lake shapes with recognizable mallard ducks:
  body, wing, white neck ring, green head, bill, eye, tail, wake, and reflection.
