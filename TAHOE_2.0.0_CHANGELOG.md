# Adventure 2 v2.0.0 — Lake Tahoe Foundation

This release creates the first playable Adventure 2 developer preview.

## Launching Tahoe

The main title screen now includes:

**Adventure 2 Developer Preview — Launch Lake Tahoe instantly**

Tahoe also opens directly at:

`tahoe.html`

Adventure 1 does not need to be completed during development.

## Included in the playable foundation

- Standalone Lake Tahoe canvas game
- Peaceful portal-arrival dialogue
- Will, Daniel, and Caleb character switching
- Existing character animation sheets
- Rigsby companion following the player
- Animated lake and water reflections
- Multi-layer mountain parallax
- Snow-capped peak
- Pine forest with healthy and early dead trees
- Granite rocks and climbable platforms
- Dirt hiking trail rising from lake level
- Wildflowers and birds
- First Tahoe Memory collectible
- Altitude and route-progress HUD
- Trailhead completion moment
- Keyboard and mobile touch controls
- Landscape mobile optimization
- Pause, restart, replay, and return-to-title controls

## Controls

- Move: Arrow keys or A / D
- Jump: Up, W, or Space
- Switch: 1, 2, 3, or SWITCH
- Pause: Escape or pause button

## Scope

There is no combat or Pine Beetle enemy system yet. v2.0.0 is intentionally the polished world and movement foundation.
